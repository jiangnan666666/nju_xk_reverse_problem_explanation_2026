# 南大选课系统验证码识别模块
## 纯技术研究 | 仅用于学习交流 | 不提供完整攻击脚本

本项目是针对**高校选课系统旋转验证码**的图像识别与坐标配对算法实现，基于 OpenCV + DDDOCR 开发，用于学习验证码识别、图像处理、字符旋转校正技术。

---

## 🔍 功能说明
本模块实现以下**技术能力**：
1. 验证码图像切割（上部字符区 / 底部提示区）
2. 字符目标检测（定位 4 个旋转字符位置）
3. **旋转识别算法**（每隔 15° 旋转一次，直到识别成功）
4. 底部字符序列与上部坐标**自动配对**
5. 线程安全的 OCR 调用
6. 鲁棒性策略：
   - 必须识别 ≥3 个字符才允许自动配对
   - 最多允许缺失 1 个字符
   - 严格校验底部 4 个不重复字符

---

## 🧠 核心技术原理


1. **图像分割**
   - 截取验证码上部字符区域
   - 截取底部明文提示区域

2. **字符检测**
   - 使用 DDDOCR 检测字符框
   - 获取每个字符的坐标

3. **旋转识别（核心）**
   - 对每个字符进行 0°~360° 旋转
   - 每次旋转 15° 进行 OCR 尝试
   - 匹配底部目标字符集

4. **坐标配对逻辑**
   - 根据底部字符顺序依次匹配上部字符中心坐标
   - 允许 1 个字符自动补全
   - 输出最终点击坐标序列

5. **线程安全**
   - 加锁保证 OCR 多线程调用不崩溃
